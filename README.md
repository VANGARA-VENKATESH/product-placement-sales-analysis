# Strategic Product Placement Analysis

This project analyzes how product placement affects sales performance in retail stores.

## Objective
To identify how shelf positioning and category placement impact revenue using data analysis and visualization.

## Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Tableau

## Features
- Data Cleaning
- Sales Trend Analysis
- Category-wise Performance
- Visualization Dashboard

## Project Structure
- data_analysis.ipynb
- sales_data.csv
- images/
- README.md

## Sample Output
Graphs showing:
- Sales by Category
- Sales by Shelf Position
- Monthly Trends
